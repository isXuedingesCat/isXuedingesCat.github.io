Linux下常用第三方源码安装

OpenBlas

  1. 下载.tra.gz压缩格式的源码
  2. 命令行进入到压缩包所在的目录中
  3. 执行 tar -zxvf OpenBLAS**.tar.gz 其中 表示版本号信息
  4. 命令行下进入加压缩后得到的目录中
  5. 依次执行以下命令 执行cmake时可以指定其他的参数，具体有那些参数可以通过查看解压后目录下的CMakeLists.txt 文件中的 option 选项，也可以直接修改 CMakeList.txt 文件中option选项的默认值，从而不必要使用cmake进行参数指定。
    执行make install 时注意权限问题，如果cmake配置时指定的目录当前用户没有操作权限会导致安装失败，需要切换用户或使用sudo make install 进行安装。

eigen

  2. 使用 tar -zxvf eigen**.tar.gz 其中 表示版本号
